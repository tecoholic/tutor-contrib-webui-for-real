webui-for-real plugin for `Tutor <https://docs.tutor.edly.io>`__
################################################################

An actual web UI for Tutor.


Installation
************

.. code-block:: bash

    pip install git+https://github.com/tecoholic/tutor-contrib-webui-for-real

Usage
*****

.. code-block:: bash

    tutor plugins enable webui-for-real
